# APPLICATION POKEMON

Création d'un application affichant une liste de Pokémon du Pokédex

## Lancement

* Avoir Android Studio

* Ouvrir le projet dans Android Studio

```
git clone https://github.com/ZephIdes/POKEMON.git
```
